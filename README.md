# WelcomeGuard

## Overview
WelcomeGuard is a face recognition application with a React/Vite/TypeScript frontend.

## Installation & Setup

1. Install dependencies:
```sh
npm install
```

2. Start the development server:
```sh
npm run dev
```

## Technologies Used

- React
- Vite
- TypeScript
- Tailwind CSS
- shadcn-ui
